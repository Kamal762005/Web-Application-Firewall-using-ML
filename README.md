# Web-Application-Firewall-using-ML
A Web Application Firewall (WAF) is a security tool that monitors and filters HTTP traffic to protect web applications from common cyberattacks like SQL injection and XSS.

## 🎯 Objectives
Build a lightweight and intelligent WAF
Detect malicious HTTP requests using ML
Improve accuracy over traditional rule-based systems
Provide real-time request filtering

## ⚙️ Tech Stack
Python
Scikit-learn
Pandas / NumPy
Flask (for web integration, if used)
Logistic Regression (ML model)

## 🧠 Machine Learning Model
Algorithm: Logistic Regression
Input: HTTP request features (URL, parameters, payload, etc.)
Output: Classification (Benign or Malicious)

## 🚀 Features
Detects common web attacks:
SQL Injection
Cross-Site Scripting (XSS)
Command Injection
Lightweight and fast prediction
Easy integration into web applications
Extendable with more ML models
